# 后续实现阶段划分

状态：阶段 1-3 已完成，阶段 4-9 待执行  
日期：2026-06-23

原则：一阶段只做一段闭环，完成后能单独验收。先让数据能保存，再让 AI 能调用，最后做汇总和优化。

## 阶段 1：账号与用户边界（已完成）

目标：让系统知道当前用户是谁。

范围：

- 注册、登录、退出。
- 找回密码和邮箱验证码的最小流程。
- 工作台页面登录后访问。
- 服务端查询统一按 `user_id` 过滤。

验收：

- 新用户能注册并进入 Dashboard。
- 退出后不能访问工作台。
- 用户只能读写自己的数据。

先不做：

- 管理后台。
- 多设备会话管理。

## 阶段 2：AI Provider 多配置（已完成）

目标：用户能保存多个自己的 AI 配置，并选择一个激活。

范围：

- 新增 `ai_provider_configs` 表。
- 保存 Provider 名称、格式、Base URL、模型、加密 API Key。
- 设置一个激活 Provider。
- 测试连接接口。

验收：

- 用户能新增多个 Provider。
- 保存不会自动激活，只有点击“设为激活”才切换。
- 后端能拿到当前激活 Provider。

先不做：

- 用量统计。
- 额度体系。

## 阶段 3：面试目标（已完成）

目标：Dashboard 和 AI 生成页共用同一个当前面试目标。

范围：

- 保存岗位、级别、技术栈、面试时间。
- Dashboard 显示当前目标。
- AI 生成页默认带入当前目标。

验收：

- 修改目标后刷新仍保留。
- AI 生成页读取同一份目标数据。

完成记录：

- 已新增当前面试目标接口、数据库迁移和 Dashboard 保存表单。
- Dashboard 与 AI 生成页已共用 `interview_targets` 当前目标。

先不做：

- 独立目标管理页。
- 多目标复杂切换。

## 阶段 4：资料上传与解析状态

目标：资料能真实上传并进入解析流程。

范围：

- 上传简历、JD、项目笔记、复习资料。
- 文件保存到本地目录或挂载目录。
- 数据库存文件元数据和路径。
- 保存解析状态、摘要、知识点、项目点、可生成方向。

验收：

- 资料列表能看到真实上传文件。
- 资料详情能看到解析状态和摘要。
- 从资料详情进入 AI 生成能带上资料来源。

先不做：

- 文件二进制入库。
- 复杂文件管理器。

## 阶段 5：AI 生成候选题

目标：从目标或资料生成候选题，并先停在确认队列。

范围：

- 调用 LiteLLM。
- 按目标生成。
- 按资料生成。
- 保存生成批次。
- AI 自动返回题型、难度、标签、答案草稿。

验收：

- 点击生成后产生真实候选题。
- 候选题刷新后仍存在。
- 生成失败有错误提示。

先不做：

- 队列系统。
- 长任务后台通知。

## 阶段 6：题库入库与管理

目标：候选题确认后进入个人题库。

范围：

- 批量确认入库。
- 题库筛选、分页、详情。
- 单题编辑。
- 归档、恢复、删除。
- 答案版本保存。

验收：

- 生成页确认入库后，题库能看到题目。
- 题库筛选条件刷新后仍正确。
- 编辑题目后详情页同步变化。

先不做：

- 独立题目编辑页。
- 手动逐题创建主流程。

## 阶段 7：练习、反馈、复习状态

目标：用户练完题后能沉淀历史、反馈和复习计划。

范围：

- 按题练习提交。
- 问答题保存文本回答。
- 选择题保存选项和判分。
- STAR/行为题复用问答反馈。
- AI 反馈保存评分、遗漏点、追问风险、建议。
- 保存错题、掌握、下次复习。

验收：

- 练习结果刷新后仍存在。
- 练习历史按时间显示。
- 复习计划能筛错题、到期题、薄弱点题。

先不做：

- 语音回答。
- 复杂学习计划。

## 阶段 8：Dashboard 真实数据

目标：Dashboard 从 mock 切到真实汇总。

范围：

- 今日待练。
- 复习到期。
- 最近练习。
- 复习压力。
- 题库、资料、AI 接入状态。
- 近 7 天练习趋势。
- 薄弱点分布。

验收：

- 生成、入库、练习后 Dashboard 数字会变化。
- 图表来自真实练习数据。

先不做：

- 图表 tooltip。
- 时间范围筛选。
- 多指标切换。

## 阶段 9：打磨与上线准备

目标：把前面真实链路稳定下来。

范围：

- 错误提示。
- 加载态。
- 空状态。
- 移动端检查。
- 基础安全检查。
- 部署文档。

验收：

- 新用户能从注册走完整链路：配置 AI -> 设目标 -> 上传资料 -> 生成题 -> 入库 -> 练习 -> 看反馈。
- 无支付、充值、套餐入口。

先不做：

- 管理后台。
- 社区题库。
- 实时面试辅助。
